用Hexo快速搭建个人博客

一、介绍

​ Hexo是一个基于 node.js的快速生成静态博客的开源框架,支持 Markdown和大多数 Octopress插件,一个命令即可部署到 Github页面、 Giteee、 Heroku等,强大的APl,可无限扩展,拥有数百个主题和插件。

二、安装Node.js

网址：Node.js (nodejs.org)

下载完毕后后打开命令行，输入node -v查看版本。返回如下图所示，表示安装成功。

三、搭建

1、安装Hexo

在命令行输入以下命令来安装Hexo：npm install hexo-cli -g

因为我之前下过，所以这里显示的是版本更新。

2、初始化博客目录

输入以下命令：hexo init xxx.github.io (这里的xxx换成你自己的英文名)，这个文件夹很重要，接下来不止一次用到，为了方便，后面都叫它目录文件夹。

3、进入目录

输入以下命令：cd xxx.github.io （这里的xxx换成你自己的英文名）

4、在该目录下安装

输入以下命令：npm install

5、生成静态页面

输入以下命令：hexo clean hexo g （g是generate，生成的意思）

6、运行网站

输入以下命令：hexo s （s是server，在服务器运行的意思）

7、登陆网站

打开浏览器，输入网址：localhost:4000

出现以上界面，说明一个简易的博客就搭建好了。

四、完善

1、修改配置

打开目录文件夹，找到配置文件（ _config.yml ），目录文件夹地址如下

用IDEA打开配置文件

修改个人信息如下，详细的配置可以看官方的配置文件Configuration | Hexo

2、选择主题

可以在官网查看各式各样的主题Themes | Hexo

这里选择NexT - Theme for Hexo (theme-next.js.org)作为演示

3、下载主题

首先下载Git (git-scm.com)

在右击目录文件夹，点击“Git Bash Here”,在弹出的窗口中输入

git clone https://github.com/iissnan/hexo-theme-next themes/next

（不同的主题下载网址不同，但是每一个主题的作者都会给出下载地址）

下载主题

4、配置主题

主题下载完之后，在你目录文件夹中的 _config.yml 文件中，修改 theme 为你的主题名字

5、运行主题

在命令行输入 Ctrl+C 停止之前的运行

接着输入npm install hexo-renderer-swig

最后依次输入hexo g hexo s 重新运行

刷新一下之前的网页，可以发现主题更改了

1、写文章

先用Typora写好，之后保存在目录文件夹的source文件中的posts文件里面。再次刷新网页，可以看见文章被上传了。
